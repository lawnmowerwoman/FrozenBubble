# Frozen Bubble Swift v0.1.1 Preview

Small hotfix for the first preview.

## Fixed

- Returning from an active game to the menu and choosing Resume now preserves the current run score, lives, board, and in-flight state.
- Resume after a fresh app start still begins from the last reached level with a fresh run score and new lives.
- Toggling colorblind mode from the menu refreshes existing bubble textures when returning to the game.
