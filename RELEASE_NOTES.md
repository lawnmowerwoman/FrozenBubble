# Frozen Bubble Swift v0.1.0 Preview

This first preview is a native macOS Swift/SpriteKit revival of the classic single-player Frozen Bubble feel.

## Highlights

- 100 original single-player levels
- Original graphics, sounds, music, and level data
- Local menu with resume, new game, sound/music toggles, colorblind mode, rushMe mode, credits, and quit
- Lives, scoring, level best scores, best run score, compressor, win/lose flow, and frozen loser animation
- Local settings and scores in `~/Library/Preferences/de.twocent.frozenbubble.plist`

## Controls

- Left / Right: aim
- Space / Up: fire / continue
- Esc: return to menu
- R: restart current level
- N: new run
- Cmd-Q: quit

## Notes

This is an early preview. The core game is playable, but packaging, layout polish, and compatibility feedback are especially welcome.
